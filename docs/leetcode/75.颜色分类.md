# 75. 颜色分类

## 题目
给定一个包含红色、白色和蓝色，一共 n 个元素的数组，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

此题中，我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。

## 示例
```
输入：nums = [2,0,2,1,1,0]
输出：[0,0,1,1,2,2]
```

## 代码实现

### javascript
```javascript
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var sortColors = function(nums) {
    var len = nums.length
    if (len < 2) {
        return nums
    }
    
    // all in [0, zero) = 0
    // all in [zero, i) = 1
    // all in [two, len-1] = 2
    var zero = 0
    var two = len
    var i = 0
    while(i < two) {
        if (nums[i] === 2) {
            two--
            swap(nums, i, two)
        } else if (nums[i] === 1) {
            i++
        } else {
            swap(nums, i, zero)
            zero++
            i++
        }
    }
};

function swap(nums, i, j) {
    [nums[i], nums[j]] = [nums[j], nums[i]]
}
```
